Web Serial Console

Connect to serial devices online, without downloading anything.

Why?

Portable

You don't need admin permission to use it, just an up-to-date version of Chrome.

Modern

Fonts, buttons, and scrollbars are custom. Dark mode included.

Keyboard-shortcut friendly

Ctrl+V pastes. Ctrl+C copies only when there's something to copy.

Open source

Hosted on GitHub Pages. Depends on the open-source xterm.js.
